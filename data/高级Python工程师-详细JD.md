# 高级Python工程师 - 详细岗位说明书

## 技术栈要求

### Python 异步编程

精通 Python 3.10+ 版本和 asyncio 异步编程，深入理解协程、事件循环、Future/Task 机制。能独立设计高并发异步任务调度方案，熟练使用 asyncio.gather、TaskGroup、信号量、队列等同步原语。理解 GIL 限制下的多进程加协程混合架构设计，有 ProcessPoolExecutor 与 asyncio 协同工作的实战经验。能编写高效的异步 HTTP 客户端，具备连接池管理、超时重试、熔断降级的设计能力。

### FastAPI 框架

3 年以上 FastAPI 实战经验，熟悉中间件机制、依赖注入系统、请求生命周期管理。能设计可复用的 Depends 依赖链，理解 request 级与 singleton 级作用域的选择策略。有自定义中间件开发经验，包括请求日志记录、限流熔断、全局异常捕获和链路追踪。熟悉 lifespan 事件管理，能在 startup 和 shutdown 阶段合理编排数据库连接池、Redis 客户端、后台任务等资源的初始化与优雅释放。

### PostgreSQL 数据库

熟练编写复杂 SQL 查询，有慢查询分析和索引优化经验。能使用 EXPLAIN ANALYZE 分析执行计划，针对多表 JOIN、聚合查询、排序操作设计联合索引或部分索引。熟悉窗口函数、CTE 递归查询、LATERAL 横向子查询等高级特性。有 Alembic 迁移工具的实战经验，能编写幂等的、可回滚的 migration 脚本，理解迁移冲突的解决策略。

### Redis 缓存

熟悉 Redis 五大核心数据结构的典型应用场景。能根据业务需求设计缓存策略，包括 Cache-Aside 旁路缓存、Write-Through 写穿、Write-Behind 写回等模式，合理设置 TTL 和淘汰策略。有分布式锁（SETNX + Lua 脚本原子释放）、限流器（滑动窗口和令牌桶算法）、延迟队列等高级场景的落地经验。

### 容器化与部署

有 Docker 生产环境部署经验，能编写规范的多阶段 Dockerfile 优化镜像体积，理解层缓存和构建上下文优化。掌握 K8s 核心资源对象的配置（Deployment、Service、ConfigMap、Secret、Ingress），理解 Pod 生命周期、健康探针、资源限制的配置策略。能编写 docker-compose 编排本地开发环境。有 Helm chart 编写经验者优先。

### ORM 与工程化

熟练使用 SQLAlchemy 2.0 异步 ORM，能设计复杂的模型关系（一对多、多对多、自引用），合理选择 lazy、selectin、joined 等加载策略以避免 N+1 查询问题。有 AsyncSession 生命周期管理经验，理解 sessionmaker 的上下文传播机制和事务边界控制。代码规范遵循 PEP 8，使用 pre-commit 钩子做自动化检查，有 CI/CD 流水线配置经验。

### 加分项技术

有 LangChain 或 LangGraph 生态开发经验者优先，包括 Agent 工具定义、工作流编排、Checkpoint 状态持久化。有钉钉开放平台 API 对接经验者优先，包括日程管理、OAuth 授权流程。有大模型 API 调用经验者优先，包括 DeepSeek、千问等模型的 Prompt 工程和输出控制。有邮件系统对接经验者优先，包括 IMAP 协议、SMTP 协议以及邮件内容的结构化解析。

## 日常工作职责

负责智能招聘平台后端核心模块的架构设计与功能开发，包括候选人管理、职位发布、简历解析、AI 评分、面试流程等模块。撰写清晰的技术设计文档，在团队内部进行方案评审和可行性论证。参与代码评审，按照团队规范把控代码质量，确保代码的可读性和长期可维护性。与算法团队协作，将 AI 能力深度集成到业务系统中，设计合理的 API 接口和数据流转方式。分析并优化系统性能瓶颈，解决高并发场景下的响应延迟和资源争用问题。编写和维护技术文档，推动团队技术规范和最佳实践的沉淀与落地。

## 软技能期待

能独立负责复杂模块的技术方案设计和落地执行，从需求分析、技术选型到上线交付全流程自闭环。具备良好的跨部门沟通能力，能与产品经理对齐需求细节，与前端同事协作定义接口契约，与算法团队探讨 AI 能力的集成方式。有较强的技术自驱力，能主动发现系统中的潜在问题并提出优化方案，而非等待问题暴露后再被动修复。习惯编写条理清晰的技术文档和代码注释，能以简洁的文字让他人快速理解设计意图和实现细节。对新技术保持好奇心和学习热情，能够将前沿技术合理应用到业务场景中。

## 业务领域与职位优势

本岗位直接参与智能招聘 SaaS 系统核心功能的开发，涉及 AI Agent 智能体、RAG 检索增强生成、大模型应用等前沿技术方向。有钉钉生态集成的实际工程场景，包括 OAuth 授权流程、日程同步、消息推送等。有邮件自动化的落地场景，包括 IMAP 轮询、SMTP 发送、邮件内容解析等。候选人在项目中可积累 AI Agent 工具编排、多模型热备降级、长链路任务状态管理、检索增强生成等高级工程经验。团队处于技术升级窗口期，有大量的架构设计和技术选型机会，技术成长空间和行业差异化竞争力明显优于同类岗位。
