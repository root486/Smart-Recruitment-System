# 公司背景与团队介绍

## 公司概况

我们是一家专注于智能招聘领域的 A 轮创业公司，团队规模约 30 人。
公司致力于通过 AI 技术提升招聘效率，让企业更快找到合适的人才。

## 技术团队

- 后端团队 5 人，前端团队 3 人，算法团队 2 人
- 技术栈以 Python 为主，前端使用 React + TypeScript
- 基础设施部署在阿里云，使用 Docker + K8s 编排
- 数据库：PostgreSQL（主库）+ Redis（缓存）
- 监控：Sentry + Prometheus + Grafana

## 团队文化

- 扁平化管理，无严格的上下级关系
- 每周一次 Code Review，注重代码质量
- 每月一次技术分享，鼓励知识沉淀
- 鼓励开源贡献，工作时间可参与开源项目

## 当前技术挑战

- 正在从单体架构向微服务架构迁移
- AI Agent 编排系统需要更好的状态管理和容错机制
- 邮箱机器人的邮件解析准确率需要持续优化
- 系统需要支持更大规模的并发处理
