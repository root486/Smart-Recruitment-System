# 公司背景与团队介绍

## 公司概况

我们是一家专注于智能招聘领域的 AI 创业公司，团队规模约 30 人，分布在产品、研发、运营三条业务线上。核心产品为智能招聘 SaaS 平台，覆盖简历自动解析、AI 多维评分、面试自动协调、入职流程管理等全链路场景，目前服务超过 200 家企业客户。

### 公司愿景

公司致力于通过 AI 技术重塑传统招聘流程，让企业用更短的时间更精准地匹配到合适的人才，同时让候选人在应聘过程中获得更高效的反馈和更透明的体验。我们的长期目标是成为国内智能招聘领域的标杆产品，以技术驱动的方式将招聘效率提升 5 倍以上。

### 发展历程

公司成立于 2022 年，创始人团队来自一线互联网公司的 HR 和 AI 实验室。2023 年初完成天使轮融资，年中推出 MVP 版本并获得首批 20 家企业客户验证。2024 年初完成 A 轮融资，目前产品已迭代至 3.0 版本，正在从单一的简历管理工具向 AI 驱动的全流程智能 HR 平台全面升级。公司当前处于快速扩张阶段，研发团队计划在年内翻倍。

## 技术团队

后端团队 5 人，负责平台核心业务系统、AI Agent 编排引擎、数据平台的建设与维护。前端团队 3 人，负责 Web 端交互体验与数据可视化。算法团队 2 人，聚焦大模型应用、NLP、RAG 检索增强等方向，与后端团队紧密协作完成 AI 能力的工程化落地。

### 技术基础设施

技术栈以 Python 为核心语言，后端基于 FastAPI 框架，前端使用 TypeScript + Vue 生态。基础设施部署在阿里云平台上，使用 Docker 容器化部署与 Kubernetes 编排调度。数据库层面采用 PostgreSQL 作为主存储引擎，通过 Alembic 管理数据库版本迁移。Redis 作为缓存层和分布式状态存储，支撑高并发场景下的低延迟访问。监控体系基于 Sentry 实现错误追踪和性能监控，结合 Prometheus + Grafana 构建系统指标采集和可视化看板，实现了从基础设施到应用层的全链路可观测。

### 工程实践

团队采用 Git Flow 分支策略，功能开发走 feature 分支，通过 Pull Request + Code Review 后合并到主分支。CI/CD 流水线基于 GitHub Actions，提交后自动触发单元测试、代码风格检查和安全扫描，通过后自动部署到预发环境。每周三为固定的发版日，采用蓝绿部署策略保证零停机发布。技术文档以 Markdown 形式存放在项目仓库的 docs 目录下，随代码一起版本管理。

## 团队文化

公司采用扁平化管理结构，没有严格的层级划分，工程师之间以技术能力而非职级互相影响决策。每位工程师都有权限直接对系统设计提出质疑和改进建议，技术方案通过 RFC 文档公开讨论后决策。

### 成长与交流

每周一次代码评审，评审不只关注代码正确性，更注重设计合理性、可维护性以及对团队规范的遵循程度。每月一次内部技术分享会，主题不限，鼓励任何有价值的输出，包括新框架调研、线上事故复盘、开源项目解读、个人技术博客分享。团队鼓励开源贡献，工作时间内允许参与外部开源项目，前提是不影响本职工作交付。每年每人有一次参加外部技术会议的机会，费用由公司承担。

### 工作环境

工程师入职即配备 MacBook Pro 和高分辨率外接显示器，办公位配置符合人体工学的升降桌和办公椅。办公区设有开放讨论区、专注工作区和休闲区，满足不同工作状态的需求。提供免费的零食饮料和现磨咖啡机。弹性工作时间，核心时段为 10:00 至 17:00，其余时间可根据个人习惯灵活安排。支持混合办公，每周最多可远程办公两天。

## 当前技术挑战

平台正从单体应用架构向微服务架构演进，需要设计合理的服务拆分边界和跨服务通信机制，涉及分布式事务、服务发现、配置中心等基础设施的建设。

### AI Agent 稳定性

AI Agent 编排系统的状态管理复杂度持续增加，多工具调用链路的错误恢复和断点续执行需要更健壮的容错机制。当前基于 LangGraph Checkpoint 的持久化方案已实现断点恢复，但在工具调用超时、LLM API 限流、部分工具不可用等场景下的降级策略仍需完善。同时需要建设 Agent 执行的可观测体系，包括每步工具调用的耗时、成功率和错误分布，以便定位长链路任务的瓶颈。

### 系统吞吐能力

系统需要支撑更大规模的并发处理，目标从单日百级候选人处理量提升到千级。当前瓶颈集中在数据库连接池的利用率、大文件简历解析的异步化、以及批量评分的并发控制策略上。计划通过引入消息队列解耦评分任务、优化缓存策略减少数据库回源、以及横向扩展 Agent 实例数量来突破吞吐天花板。

## 薪酬福利

我们提供有竞争力的现金薪酬和全面的福利保障，同时注重员工的长期成长和工作生活平衡。

### 薪酬结构

现金薪酬结构为 12 个月底薪 + 2-4 个月年终奖金，每年两次调薪评估窗口。早期员工享有期权激励，行权期为 4 年，首年 cliff 后按月 vest。五险一金按实际工资基数缴纳，公积金按最高比例 12% 缴存。公司额外购买补充商业医疗保险，覆盖门诊和住院报销，配偶和子女可自费加入。

### 假期与福利

年假制度为入职即享 10 天带薪年假，每工作满一年增加 2 天，上限 20 天。每年一次海外团建，费用公司全额承担。提供年度学习预算 5000 元，可用于购买技术书籍、在线课程平台订阅和参加行业技术会议。办公区常备零食饮料，每周五下午为自由学习时间，不安排任何会议。
